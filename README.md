# SteveTheJobs
Calamity mod add-on (Terraria), that adds a new pet - Steve Jobs
## How to install
1. Download *SteveTheJobs.tmod* file from releases on github
2. Go to below path
```
Terraria\tModLoader\Mods
```
3. Put downloaded file here
